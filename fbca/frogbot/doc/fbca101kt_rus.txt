Описание
===========
Quake mod, который комбинирует Frogbot, Clan Arena и Kombat Teams (KTPro).


Инсталляция
============
Создайте папку Fbca и распакуйте в неё архив.

Запуск игры
================
Откройте папку fbca и создайте файл fbca.bat со следующим содержанием:
---------
@echo off
cd..
qwsv.exe +gamedir fbca
---------
Потом запускаете qwcl.exe и соединяетесь к локальному серверу.
Для игры рекомендую использовать ezQuake.exe или fuhquake.exe.
Тогда пишете так:
---------
@echo off
cd..
(ezQuake или fuhquake).exe +gamedir fbca
---------

Важные замечания
================
Читайте файлы readme.txt, readme012c.txt для более подробной информации о Frogbot.
Такие файлы имеются в стандартных Frogbots.
Вместо перезагрузки карты можно использовать команду "reset" для изменения настроек таких,
как  "deathmatch", "arena", и т.д. 
Если после смерти модель бота становится невидимой используйте "localinfo proxy 1".
"setinfo lw #" используйте для изменения позиции на экране +wp_stats.
Например: setinfo lw 30 - это переместит +wp_stats на экране на 30 линий ниже.

Стандартные команды

rl_pref		: бот чаще использует rl
lg_pref	: бот чаще использует shaft

Как изменить имя команды бота!
=============================
В консоле сервера набираете:
------------------------
localinfo team(N) (имя)
------------------------
Где N - это номер команды от 0 до 13.
Пример: localinfo team4 red

Все названия команд можно прописать заранее в *.cfg файлах (server.cfg, duel.cfg и т.д.)
Если вы используете fuhquake или zquake то менять имена команд командой localinfo можно
прямо в консоле (при условии что используется внутренний сервер).

Назначения со стороны юзера
=========================== 
setinfo lw 0				// позиция +wp_stats на экране

Назначения со стороны сервера
============================= 
localinfo k_timetop 30		// максимальное время в минутах регулируемое игроком для игры
localinfo k_overtime 2		// overtime режим (0 = выкл, 1 = время, 2 = до первой смерти)
localinfo k_exttime 3			// продолжительность overtime, в режиме overtime == 1
localinfo k_count 10			// количество секунд для countdown
localinfo d_skill 20			// изначальный skill
localinfo allow_startsound 1		// стартовый звук в режиме match (0 = выкл, 1 = вкл)
localinfo k_spectalk 0		// зрители могут переписываться с игроками в течение игры
localinfo add_q_aerowalk 1		// quad на карте aerowalk (0 = выкл, 1 = вкл)
localinfo k_kfp 1			// защита от частого использования команды "kill"
localinfo ffa_mapqueue 1		// карты в случайном порядке для ffa режима (0 = выкл, 1 = вкл)
localinfo autoreport 0		// авто сообщения для команды от человека (0 = выкл, 1 = вкл)
localinfo proxy 0			// эмуляция модели игрока под qizmo или mqwcl (0 = выкл, 1 = вкл)
localinfo k_motd1 "fbca"		// информация в motd
localinfo k_motd2 "admin"		// информация в motd
localinfo team0 zero			// имя команды 0
localinfo team1 one			// имя команды 1
localinfo team2 rock			// имя команды 2
localinfo team3 cure			// имя команды 3
localinfo team4 red			// имя команды 4
localinfo team5 push			// имя команды 5
localinfo team6 ram			// имя команды 6
localinfo team7 star			// имя команды 7
localinfo team8 none			// имя команды 8
localinfo team9 mesh		// имя команды 9
localinfo team10 jag			// имя команды 10
localinfo team11 fore		// имя команды 11
localinfo team12 edge		// имя команды 12
localinfo team13 blue		// имя команды 13
